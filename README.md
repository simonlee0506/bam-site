# BAM 文献助手 — 官网

面向生物、农业与医学研究者的多源文献检索 + AI 综述工具。

支持 PubMed、Europe PMC、bioRxiv/medRxiv、OpenAlex 一键检索，提供 AI 综述生成、全文翻译、摘要翻译、批量 PDF 下载等功能。
